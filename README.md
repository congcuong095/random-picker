# Random Picker

A collection of fun, interactive random picker tools built with vanilla HTML, CSS, and JavaScript.

**Live Demo:** [random-picker.online](https://random-picker.online)

## Tools

| Tool | Description |
|------|-------------|
| **Random Card** | Draw random cards from a standard deck with flip animation |
| **Random Number** | Generate random numbers with custom range, count, and elimination mode |
| **Random Spinner** | Spin the wheel with custom items and smooth easing animation |
| **Random Dice** | Roll up to 6 dice with shake animation and total calculation |
| **Random Coin** | Flip a coin with realistic toss animation and history tracking |
| **Name Picker** | Pick a random name from your list with auto-save and elimination mode |
| **Random Color** | Generate random colors with HEX & RGB values, copy with one click |

## Tech Stack

- HTML / CSS / JavaScript
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS framework
- [Font Awesome](https://fontawesome.com/) — Icons
- [Animate.css](https://animate.style/) — CSS animations
- [SweetAlert2](https://sweetalert2.github.io/) — Popup dialogs
